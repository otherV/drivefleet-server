# 🚗 DriveFleet

### A Premium Full-Stack Car Rental Platform

🌐 **Live Site:** [DriveFleet](your-vercel-url-here)

---

## ✨ Features

- 🔐 **Secure Authentication** — Email/password and Google OAuth via Better Auth with JWT protection
- 🚗 **Browse & Search** — Explore all available cars with search by name and filter by type
- 📅 **Instant Booking** — Book cars with driver options, special notes and real-time availability check
- 🛠️ **Car Management** — Add, update and delete your own car listings with full CRUD support
- 📋 **Booking Dashboard** — View and cancel your bookings anytime from your personal dashboard
- 🔒 **Enterprise Auth** — JWT-protected Express API routes with JWKS verification
- 🌙 **Theme Toggle** — Dark/light mode with persistent localStorage preference
- 📱 **Fully Responsive** — Optimized for mobile, tablet and desktop
